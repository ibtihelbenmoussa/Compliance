- Use images for diagrams and charts
- Compress large files before uploading

## Technical Notes

- Files are uploaded as part of form submission (not AJAX)
- All files sent in single request (multipart/form-data)
- Server validates each file independently
- Failed validation rolls back entire transaction
- Files stored with UUID names to prevent conflicts
- Original filenames preserved in database
